Use of two hands: **hand1** ja **hand2** **write** command will set the angle of the hand (in degrees). Command moving the hand to that orientation. Eg. move the hand to 90 degrees orientation: ``` hand1.write(90); ``` --- **delay** command pauses the program for the amount of time (in miliseconds) specified as parameter. Eg. wait two seconds: ``` delay(2000); ```
